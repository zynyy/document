docker container run

描述(Description)#

创建一个容器并且启动运行

用法(Usage)#

 docker container run [OPTIONS] IMAGE [COMMAND] [ARG...]

相同功能命令(alias)#

  1. docker run

标注详解#

  1. Cgroups 是control groups缩写有7个子系统组成 cpuset、cpu、cpuacct、blkio、devices、freezer、memory了解更多这些控制组都在/sys/fs/Cgroups目录下面↩︎

  2. CFS(Completely Fair Scheduler) 完全公平调度器了解更多↩︎

  3. 搜索域查看官方解答↩︎

  4. mount 的类型可以是volume、bind、tmpfs详细了解↩︎

  5. 标准流有: 标准输入stdin、标准输出stdout、标准错误、stderr↩︎
