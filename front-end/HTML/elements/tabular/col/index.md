# `col` 元素标签

## 用法

```html
// 注意: 一个 <table> 元素中 <tr> 元素存在几个子元素 <td> 则该标签就存在几个
<table>
  <colgroup>
    <col />
  </colgroup>
  <tr>
    <td></td>
  </tr>
</table>
```

## 子元素

> 该标签不存在任何子元素和文本内容
