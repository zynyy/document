kbd 元素标签

键盘输入组合

用法#

// 注意: 如果存在子元素 <samp> 则表示系统输出。如果父元素是 <samp> 则表示输入，子元素 <kbd> 则代表键盘组合键
<kbd>
  子元素 or 文本内容
</kbd>

子元素标签#

  1. metadata: <link rel=''>(属性rel必须存在)

  2. scripting: <script>、<noscript>、<canvas>、<template>

  3. emabedded: <audio>、<video>、<img>、<map>、<iframe>、<embed>、<object>、<picture>

  4. edit: <ins>、<del>

  5. textLevel: <a>、<abbr>、<b>、<bdi>、<bdo>、<br>、<cite>、<code>、<data>、<dfn>、<em>、<i>、<kbd>、<mark>、<q>、<ruby>、<s>、<samp>、<small>、<span>、<strong>、<sub>、<sup>、<time>、<u>、<var>、<wbr>

  6. forms: <button>、<input>、<meter>、<output>、<progress>、<select>、<textarea>、<label>、<datalist>

  7. SVGElement: <svg>

  8. MathMLElement: <math>

CSS 默认样式#

kbd {
  font-family: monospace;
}
