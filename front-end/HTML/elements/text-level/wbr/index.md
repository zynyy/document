# `wbr` 元素标签

换行

```html
// 注意：通常紧跟在文章的符号后面。如果 父级元素 CSS 样式含有 work-break 属性则该标签无效
<wbr />
```

## 子元素标签

> 该标签不存在任何子元素和文本内容

## 属性

* [全局属性](/front-end/HTML/attribute#anchor-全局属性)
