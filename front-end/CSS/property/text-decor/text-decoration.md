# text-decoration 文本线条

该属性为简写属性(3种)，给元素文本内容装饰线条。3种属性分别为: `text-decoration-line`、`text-decoration-style`、`text-decoration-color`。

## 语法

```css
text-decoration: [
  <text-decoration-line>
  || <text-decoration-style>
  || <text-decoration-color>
]{1, 3}

eg:
text-decoration: none;
```
