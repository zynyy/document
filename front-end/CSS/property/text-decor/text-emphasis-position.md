# text-emphasis-position 标记位置

给元素文本内容重点标记设置位置

## 语法

```css
text-emphasis-position: [
  [
    over
    || under
  ]
  && [
    right
    || left
  ]?
]

eg:
text-emphasis-position: over;
text-emphasis-position: over right;
```
