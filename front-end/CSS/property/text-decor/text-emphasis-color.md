# text-emphasis-color 标记颜色

给元素文本内容标记设置颜色

## 语法

```css
text-emphasis-color: [
  <color>
]

eg:
text-emphasis-color: red;
```

## 属性值

* [全局属性值](/front-end/CSS/values#anchor-值类型)
* [color](/front-end/CSS/values#anchor-值类型)
