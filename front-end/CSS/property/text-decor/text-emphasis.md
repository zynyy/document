# text-emphasis 文本标记

该属性为简写属性(2种)，给元素文本内容设置重点标记。2种属性分别为: `text-emphasis-style`、`text-emphasis-color`;

## 语法

```css
text-emphasis: [
  <text-emphasis-style>
  || <`text-emphasis-color>
]{1, 2}

eg:
text-emphasis: red;
```
