# text-decoration-color 线条颜色

给元素文本内容的线条设置颜色

## 语法

```css
text-decoration-color: [
  <color>
]

eg:
text-decoration-color: red;
```

## 属性值

* [全局属性值](/front-end/CSS/values#anchor-值类型)
* [color](/front-end/CSS/values#anchor-值类型)
