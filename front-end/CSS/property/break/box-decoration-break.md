# box-decoration-break 多行文本样式属性使用

对元素属性(7种)在多行文本内容如何使用，7种分别为属性: `background` 、`padding`、`margin`、`border` 、`border-image`、`box-shadow`、`clip`。

## 语法

```css
box-decoration-break: [
  slice
  || clone
]

eg:
box-decoration-break: slice;
```
