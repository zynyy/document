# text-align-last 文本段落最后一行对齐

给元素文本内容中每一段的最后一行设置对齐方式

## 语法

```css
text-align-last: [
  auto
  || start
  || end
  || left
  || right
  || center
  || justify
  || match-parent
]

eg:
text-align-ast: start;
```
