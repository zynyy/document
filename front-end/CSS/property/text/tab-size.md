# tab-size 制表符(U+0009)的宽度

设置`tab`键制表符宽度

## 语法

```css
tab-size: [
  <integer>
  || <length>
]

eg:
tab-size: 4;
```

## 属性值

* [全局属性值](/front-end/CSS/values#anchor-值类型)
* [integer](/front-end/CSS/values#anchor-值类型)
* [length](/front-end/CSS/values#anchor-值类型)
