# white-space 文本空白符

处理元素文本内容中的空白符号以及换行

## 语法

```css
white-space: [
  normal
  || pre
  || nowrap
  || pre-wrap
  || break-spaces
  || pre-line
]

eg:
white-space: break-spaces;
```
