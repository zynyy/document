# word-wrap 长单词换行

防止元素文本内容溢出元素框，是否允许单词中间断行(即一个完整单词一半在上一行一半在下一行),如果要想单词完整的在一行可以使用HTML标签中的`<wbr>`; 在 CSS3 中改名为 `overflow-wrap`。

## 语法

```css
overflow-wrap: [
  normal
  || break-word
]

eg:
overflow-wrap: break-word;
```
