# hyphens 换行是否需要连字符(-)连接

当元素中的文本内容(单词)进行自动换行时，如果出现上下行各有一些时是否需要连字符进行连接。连字规则具有语言特定性。所以是由 HTML 元素中的 `lang` 属性值决定的。

```css
hyphens: [
  none
  || manual
  || auto
]

eg:
hyphens: manual;
```
