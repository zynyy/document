# text-align-all 文本段落每一行对齐方式一样

给元素文本内容设置对齐方式。请使用 `text-align`来代替此属性

## 语法

```css
text-align-all: [
  start
  || end
  || left
  || right
  || center
  || justify
  || match-parent
]

eg:
text-align-all: center;
```
