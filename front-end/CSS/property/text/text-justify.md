# text-justify 行对齐

当元素设置文本对齐(`text-align`)方式为`justify`时，使用该属性进行行对齐。

## 语法

```css
text-justify: [
  auto
  || none
  || inter-word
  || inter-character
]

eg:
text-justify: auto;
```
