# initial-letters-wrap

## 语法

```css
initial-letters-wrap: [
  none
  || first
  || all
  || grid
  || <length>
  || <percentage>
]

eg:
initial-letters-wrap: none;
```

## 属性值

* none
* first
* all
* grid
* [length](/front-end/CSS/values#anchor-值类型)
* [percentage](/front-end/CSS/values#anchor-值类型)
