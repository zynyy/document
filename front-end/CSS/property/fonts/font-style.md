# font-style 字体样式

控制元素文本内容的字体族样式

## 语法

注: []表示一组可选的属性值，|| 表示 [] 内的属性值任选其一。

```css
font-style: [
  normal
  || italic
  || oblique
]

eg:
font-style: italic;
```
