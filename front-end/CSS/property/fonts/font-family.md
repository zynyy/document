# font-family 字体族名

给元素文本内容设置字体族

## 语法

```css
font-family: [
  <family-name-string>
  || <generic-family>
]

eg:
font-family: serif;
font-family: 'Microsoft YaHei';
/* 可以设置多个 字体列表用英文逗号(,)隔开*/
font-family: sans-serif, serif;
```

## 属性值

* generic-family:如下
  1. serif
  1. sans-serif
  1. cursive
  1. fantasy
  1. monospace
