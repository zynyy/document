# font-kerning 字母间距

控制元素文本内容(字母)的间距

## 语法

```css
font-kerning: [
  auto
  || normal
  || none
]

eg:
font-kerning: normal;
```

* [全局属性值](/front-end/CSS/values#anchor-值类型)
