# font-variant 英文字母变体

该属性为简写属性(5种),给元素文本内容(含有英文字母)设置大小写并且以缩小形式。5种属性分别为: `font-variant-ligatures`、`font-variant-position`、`font-variant-caps`、`font-variant-numeric`、`font-variant-east-asian`;

## 语法

```css
font-variant: [
  normal
  || none
  || [
    <ligatures>
    || <position>
    || <caps>
    || <numeric>
    || <east-asian>
  ]
]
```
