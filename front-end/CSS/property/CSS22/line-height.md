# line-height 行高

## 语法

```css
line-height: [
  normal
  || <number>
  || <length>
  || <percentage>
]

eg:
line-height: 1.5em;
```

## 属性值

* `normal`:
* [number](/front-end/CSS/values#anchor-值类型)
* [length](/front-end/CSS/values#anchor-值类型)
* [percentage](/front-end/CSS/values#anchor-值类型)
