# background-origin 背景图像的定位区域

## 语法

注: []表示一组可选的属性值，|| 表示 [] 内的属性值任选其一。

```css
background-origin: [
  padding-box
  || border-box
  || content-box
]

eg:
background-origin: padding-box; /* 元素背景图象宽度、高度为 10px */
```
