# border-bottom-color 边框线颜色

给元素设置下边框线颜色。

## 语法

```css
border-bottom-color: [<color>]

eg:
border-bottom-color: red; /* 设置下方向边框线颜色为 red */
```

## 属性值

1. [全局属性值](/front-end/CSS/values#anchor-值类型)
1. [color](/front-end/CSS/values#anchor-值类型)
