# border-top-color 边框线颜色

给元素设置上边框线颜色。

## 语法

```css
border-top-color: [<color>]

eg:
border-top-color: red; /* 设置上方向边框线颜色为 red */
```

## 属性值

1. [全局属性值](/front-end/CSS/values#anchor-值类型)
1. [color](/front-end/CSS/values#anchor-值类型)
