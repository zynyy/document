# background-clip 背景图像裁剪区域

## 语法

注: []表示一组可选的属性值，|| 表示 [] 内的属性值任选其一。

```css
background-clip: [
  border-box
  || padding-box
  || content-box
]

eg:
background-clip: border-box; /* 元素背景图象宽度、高度为 10px */
```
