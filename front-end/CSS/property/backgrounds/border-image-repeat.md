# border-image-repeat

## 语法

注: []表示一组可选的属性值，|| 表示 [] 内的属性值任选其一，{} 表示 [] 内的属性值任选其中几个属性值并且用空格隔开。

```css
border-image-repeat: [
  stretch
  || repeat
  || round
  || space
]{1, 2}

eg:
border-image-repeat: stretch;
```
