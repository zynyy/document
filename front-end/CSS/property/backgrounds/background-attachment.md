# background-attachment 背景图像是否固定

给元素设置背景图像是随对象内容滚动还是固定的。

## 语法

注: []表示一组可选的属性值，|| 表示 [] 内的属性值任选其一。

```css
background-attachment: [
  scroll
  || fixed
  || local
]

eg:
background-attachment: scroll; /* 元素背景图象宽度、高度为 10px */
```
