# background-color 背景颜色

给元素设置背景颜色。

## 语法

注: []表示一组可选的属性值。

```css
background-color: [
  <color>
]

eg:
background-color: red; /* 设置 背景颜色为 red; */
```

## 属性值

1. [全局属性值](/front-end/CSS/values#anchor-值类型)
1. [color](/front-end/CSS/values#anchor-值类型)
