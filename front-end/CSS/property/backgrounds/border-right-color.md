# border-right-color 边框线颜色

给元素设置右边框线颜色。

## 语法

注: []表示一组可选的属性值。

```css
border-right-color: [
  <color>
]

eg:
border-right-color: red; /* 设置右方向边框线颜色为 red */
```

## 属性值

1. [全局属性值](/front-end/CSS/values#anchor-值类型)
1. [color](/front-end/CSS/values#anchor-值类型)
