# border-color 边框线颜色

该属性为简写属性(4种),给元素设置边框线颜色。4种属性分别: `border-top-color`、`border-right-color`、`border-bottom-color`、`border-left-color`。

## 语法

注: []表示一组可选的属性值。

```css
border-color: [
  <color>
]

eg:
border-color: red; /* 设置 4 个方向边框线颜色为 red */
```
